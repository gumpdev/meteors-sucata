# meteors-sucata
A example game of meteors on sucata game engine

## Getting started

1. Install [Sucata Engine](https://github.com/gumpdev/sucata) on your computer
2. Then, you can run the project using: `sucata run .`

> You can build the game with `sucata build .`

## How to play

### Objective
Don't let the meteors reach the end of the screen, uses your ship to shoot on the meteor and destroys them.

### Controls 

- Uses arrows or A and D to move the ship
- Use Space bar to shoot
